# Ludus
#### Communication Platform


## Dependencies

* Ruby 2.1.5
* Rails 4.1.6
* MySQL

## Installation

```     
bundle install
rake db:setup
rake db:migrate
```

## Local Testing

Start Rails Server

```
rails s
```

Start MySQL Server

```
mysql.server start
```
